Click within either the lower or upper range value to overide the default selection behavior.

Entering a value on the left while leaving the right empty represents a filter matching all values greater than or equal to the supplied value. Conversely, entering a value on the right and leaving the left empty represents a filter matching all values less than or equal to the supplied value. Entering values in both boxes will match all values within the range specified. The opposite applies when not between is selected from the choices above.

Clicking on the histograph will fill the range limits. If a range limit has been selected (by clicking in it before clicking on the graph), that limit will receive the value. If no limit is selected, an empty limit will receive the value (lower first, then upper). If both limits are defined, the upper limit will be updated.
